# 🎮 Jogo da Forca em PHP

## 📌 Sobre o Projeto
Este é um projeto acadêmico desenvolvido com o objetivo de praticar e aplicar conceitos básicos de **PHP**. Trata-se de uma versão web do clássico "Jogo da Forca", focado na interação entre o frontend e o backend. 

A interação do jogador é feita através de um formulário HTML simples (input de texto), que envia a letra escolhida para o servidor. O backend em PHP é responsável por processar a tentativa, verificar se a letra existe na palavra secreta e atualizar o estado da partida.

## 🚀 Estado Atual do Desenvolvimento
O projeto encontra-se em **fase inicial de desenvolvimento** (Work in Progress). 

**Atualmente implementado/em andamento:**
* Estruturação básica do formulário HTML para receber o input do jogador.
* Lógica inicial de backend em PHP para processar os dados enviados pelo formulário.

**Próximos passos previstos:**
* Gerenciamento de estado (sessões em PHP) para manter a palavra oculta, as letras já tentadas e o número de vidas/erros entre os reloads da página.
* Lógica de vitória e derrota.
* Feedback visual simples para mostrar a forca e a palavra sendo revelada.

## 🛠️ Tecnologias Utilizadas
* **PHP:** Processamento da lógica do jogo, manipulação de strings e controle de estado.
* **HTML:** Estruturação da interface do usuário e formulário de input.
